Supporting

The NIST Viral Records Death Reporting Test Suite supports testing in support of the Vital Record Community, including structure and message content validation in accordance with the HL7 Version 2.6 Implementation Guide: Vital Records Death Reporting, Release 1 - US Realm, Standard for Trial Use August 2016 interoperability standard.

Supported Browsers

The following browsers are supported:

  • Chrome (Recommended)
  • Firefox (Recommended)
  • Safari
  • IE 9 and 10
